Ablator Process

Introduction

Ablator is a distributed experiment execution framework for ablation studies.

Step 1: Iterate initial state.


Adopt trained meta-model to iterate initial state.

Step 2: Assign resources


Utilize features from initial state to assign resources and conduct persistent experiment.

Step 3: Generate artifacts analysis


Use multiple results with fault tolerance to generate artifacts analysis.

Step 4: Revise meta-model


Revise meta-model by artifacts analysis and keep iterating initial state.
